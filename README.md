# Simple Calculator (Console Application)

## Description
This is a simple Java console-based calculator that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User input using Scanner
- Menu-driven program using switch-case

## Technologies Used
- Java
- VS Code
- JDK

## How to Run

1. Compile the program:

```bash
javac Calculator.java
```

2. Run the program:

```bash
java Calculator
```

## Sample Output

```
Enter first number: 10
Enter second number: 5

Choose operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter your choice: 1

Result = 15
```
